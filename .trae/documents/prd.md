## 1. 产品概述
长租公寓运营管理系统——面向公寓运营人员的全流程数字化管理平台，统一管理房源、租客和日常服务，解决信息分散、流程脱节、数据不透明等痛点。
- 目标用户：长租公寓运营团队（店长、管家、维修师傅、财务人员）
- 核心价值：提升运营效率、降低空置率、规范费用收缴、量化服务品质

## 2. 核心功能

### 2.1 用户角色
| 角色 | 进入方式 | 核心权限 |
|------|----------|----------|
| 店长/管理员 | 账号密码登录 | 全部功能，含报表与系统设置 |
| 管家 | 账号密码登录 | 房源看板、租约管理、费用中心、维修工单 |
| 维修师傅 | 账号密码登录 | 维修工单（查看、处理、回访） |
| 财务人员 | 账号密码登录 | 费用中心、报表页 |

### 2.2 功能模块
1. **房源看板**：空置房源、即将到期房源、欠费房源、配置状态总览
2. **租约管理**：租客资料录入、合同周期管理、押金管理、合租人管理、续租记录
3. **费用中心**：租金/水电/物业账单生成、标记收款、费用减免、催缴通知
4. **维修工单**：报修登记、分派师傅、进度跟踪、回访记录
5. **报表页**：入住率、收缴率、维修耗时、待办清单（按房间/楼栋/月份维度）

## 3. 核心流程

**租约全生命周期流程**：
运营人员在房源看板发现空置房源 → 录入新租约（租客资料、合同周期、押金）→ 合同生效后自动生成租金账单 → 租客缴费后标记收款 → 合同到期前提醒续租或退租 → 退租后房间回到空置状态。

**维修工单流程**：
租客报修/管家登记 → 创建工单 → 分派维修师傅 → 师傅处理并更新进度 → 完成维修 → 管家回访租客 → 工单关闭。

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**：深蓝灰 #1E293B 为底色，搭配翠绿 #10B981 作为操作强调色，琥珀 #F59E0B 作为警示/提醒色
- **辅助色**：石板灰 #64748B 用于次要文字，浅灰 #F1F5F9 用于卡片背景，红色 #EF4444 用于欠费/紧急状态
- **按钮风格**：圆角 8px，主操作按钮实心翠绿底白字，次操作按钮浅灰底深色字，危险操作红色底
- **字体**：标题使用 Noto Sans SC Bold，正文使用 Noto Sans SC Regular，数据使用 JetBrains Mono
- **布局风格**：左侧固定导航栏 + 顶部面包屑 + 右侧主内容区，卡片式布局
- **图标风格**：线性图标，24px，2px 线宽，与文字等高对齐
- **圆角体系**：大卡片 12px，按钮/输入框 8px，标签 6px

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 房源看板 | 状态统计卡 | 4 张等宽统计卡，顶部图标 + 数字 + 同比趋势箭头，悬停微缩放动画 |
| 房源看板 | 房源列表 | 表格布局，状态列使用彩色标签，行悬停高亮，支持分页和筛选栏 |

| 报表页 | 图表区域 | 折线图/柱状图 + 数据表格联动，图表区域白色卡片背景，微阴影 |

### 4.3 响应式设计
- 桌面优先（≥1280px），完整功能体验
- 平板适配（768px-1279px），侧边栏收窄为图标模式，表格横向滚动
- 移动端（<768px），侧边栏变为底部标签栏，卡片单列堆叠

### 4.4 动效设计
- 页面切换：淡入 + 轻微上移 8px
- 卡片悬停：阴影加深 + 上移 2px
- 数据加载：骨架屏占位
- 状态变更：颜色渐变过渡 300ms
- 模态弹窗：背景模糊 + 弹窗缩放进入
